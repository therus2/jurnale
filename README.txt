Проект Flutter: ip152_schedule_v2

Структура:
lib/
 - main.dart           (главное меню, показывает чётность недели и кнопку 'Открыть сегодня')
 - pair_detail.dart    (экран для добавления/просмотра заметок)
 - monday.dart ... sunday.dart  (по одному экрану на день)

assets/data/*.datrt — файлы с расписанием (JSON)

Инструкции для запуска:
1) Распакуй архив.
2) flutter pub get
3) Запусти приложение на устройстве/эмуляторе.

Заметки:
- Заметки сохраняются в SharedPreferences и привязываются к ближайшей будущей паре того же предмета.
- В каждом day-файле используется фильтрация по чётности недели, как и просил.
